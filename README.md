# Docker Base Image - Native Builder for C/C++/Rust

This image would normally be used for multi-stage `docker` build (as `builder`)

## Builder Versions

- GCC/G++ 9.2.1 (Stable)
- Clang 10.0.0 (Stable)
- Rust 1.42.0 (Stable)
